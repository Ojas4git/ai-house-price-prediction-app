# 🏠 House Price Prediction App

This project is a Machine Learning-based web application that predicts house prices based on various features.

---

## 🚀 Features
- Predicts house prices based on user inputs
- Uses Random Forest Regression model
- Handles real-world dataset with missing values and feature engineering
- Interactive web interface using Streamlit
- Deployed on Streamlit Cloud

---

## 📊 Model Performance
- RMSE: ~28,700
- Model: Random Forest Regressor

---

## 🛠️ Tech Stack
- Python
- Pandas, NumPy
- Scikit-learn
- Streamlit

---

## 🌐 Live Demo
https://ai-house-price-prediction-app.streamlit.app

## 📁 Project Structure
- app.py → Streamlit application
- house_model.pkl → trained model
- columns.pkl → feature columns
- requirements.txt → dependencies
